Class PCL_Pattern

PCL_Pattern

Make a pattern request to the ColourLovers API.

PCL_Pattern is almost the same as PCL_Patterns except for the fact that there are no optional parameters to specify. Setting $patternID in the constructor will request the data for that pattern id. Run PCL_Pattern::get() to request the data.
