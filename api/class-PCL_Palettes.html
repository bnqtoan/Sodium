Class PCL_Palettes

PCL_Palettes

Make a palettes request to the ColourLovers API.

The PCL_Palettes subclass is a set of methods for setting parameters alloed when calling the api/palettes portion of the ColourLovers API. Running PCL_Palettes::get() will send the request to the API and return the data formatted in an array.
