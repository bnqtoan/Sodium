Class PCL_Colors

PCL_Colors

Make a colors request to the ColourLovers API.

The PCL_Colors subclass is a set of methods for setting parameters alloed when calling the api/colors portion of the ColourLovers API. Running PCL_Colors::get() will send the request to the API and return the data formatted in an array.
