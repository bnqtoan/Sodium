Class phpColourLover

phpColourLover

Make requests to the COLOURLovers API.

ColourLovers released the COLOURLovers API on April 2nd, 2008. I wanted a PHP wrapper to this API so I wrote phpColourLover. This is an interface for the COLOURLovers API that enables you to make any request that the API allows. You can get Color(s), Palette(s), Pattern(s) and Lover(s) through the API using cURL requests. The XML response is parsed into an array using SimpleXML. The reason I don't just return the SimpleXML object is because personally I do not like how the class is set up.
